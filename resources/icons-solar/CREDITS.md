# "Solar" colored toolbar icon set — attribution

This is one of two "Colored icons" options for the toolbar (Settings > Preferences >
General > Toolbar icon style), alongside the default line-icon set in `resources/icons/`
and the sibling "IconPark" colored set in `resources/icons-iconpark/`.

**Solar Icons** (Bold Duotone style) — © 480 Design, distributed via
[Iconify](https://icon-sets.iconify.design/solar/) / <https://github.com/480-Design>.
Licensed under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/).

Solar's Bold Duotone icons achieve their two-tone look via a single `currentColor` fill
with `opacity=".5"` on the lighter layer — so each icon here has that token replaced with
one fixed colour, **Open Color green-8 (`#2f9e44`)**, chosen to read clearly on both light
and dark toolbar backgrounds (this project's line-icon set already uses the same shade as
an accent colour, e.g. in `resources/icons/playback.svg`). No other modifications were
made beyond selecting icons and this one colour substitution.

Source data: `solar.json` from <https://github.com/iconify/icon-sets> (Iconify's
normalized re-export of the original icon set).
