# "Fatcow" colored toolbar icon set — attribution

This is one of two "Colored icons" options for the toolbar (Settings > Preferences >
General > Toolbar icon style), alongside the default line-icon set in `resources/icons/`
and the sibling "Fugue" colored set in `resources/icons-fugue/`.

**Fatcow Farm-Fresh Web Icons** — © FatCow Web Hosting, <https://www.fatcow.com/free-icons>
(the original site is gone; mirrored at <https://github.com/gammasoft/fatcow>, which
preserves the original credit text). Licensed under
[CC BY 3.0](https://creativecommons.org/licenses/by/3.0/).

`function-list.png` is `document_tree.png`, `func-node.png`/`func-leaf.png` are both
`node.png`, and `record.png` is `bullet_red.png` — Fatcow has no purpose-built icons for
these Notepad++-specific concepts, so the closest available Fatcow icons were reused.
There is no Fatcow pushpin icon at all; the tab-pin button falls back to the default line
icon when this style is active (the app's icon loader falls back automatically for any
name a colored set doesn't provide).

No modifications were made to the individual icon artwork beyond selecting and renaming
files to match this project's icon-name conventions.
